The Language Reference is the lookup half of this manual.

Use it when you need the current contract quickly:

- which forms the lexer and parser recognize
- which builtin types exist
- which expressions and statements are modeled today
- which annotations and construct rules have compiler meaning
- which features are parsed, checked, or runnable
